# Система управления рейсами и бронями авиакомпании

Веб-приложение для управления рейсами и бронями билетов в личном кабинете сотрудника авиакомпании.

## Технологии

- **Клиентская часть**: HTML, CSS, JavaScript (Vanilla)
- **Серверная часть**: Node.js, Express.js
- **База данных**: MySQL

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте базу данных в MySQL Workbench:
   - Откройте MySQL Workbench
   - Создайте новую базу данных с именем `airline_db`
   - Откройте файл `database.sql` и выполните SQL скрипт для создания таблиц и заполнения тестовыми данными

3. Настройте подключение к базе данных:
   - Откройте `db-config.js` и укажите параметры подключения к вашей MySQL базе данных:
     ```javascript
     module.exports = {
         host: 'localhost',
         user: 'root',
         password: 'ваш_пароль', // Замените на ваш пароль MySQL
         database: 'airline_db',  // Название вашей базы данных
         waitForConnections: true,
         connectionLimit: 10,
         queueLimit: 0
     };
     ```

4. Запустите сервер:
```bash
npm start
```

5. Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## Функциональность

### Управление рейсами
- Просмотр списка рейсов
- Добавление нового рейса
- Редактирование информации о рейсе
- Удаление рейса
- Проверка уникальности рейсов (пункт назначения + дата/время вылета)

### Управление бронями
- Просмотр броней для каждого рейса
- Добавление новой брони
- Редактирование брони
- Удаление брони
- Перенос брони на другой рейс (только на рейсы с тем же пунктом назначения)
- Проверка ограничений (один человек - одно место на рейсе)
- Проверка доступности мест

### Параметры рейса
- ID (автоматически генерируется, нередактируемый)
- Дата и время вылета
- Пункт назначения
- Самолет (выбор из предустановленного списка)

### Параметры брони
- ID (автоматически генерируется, нередактируемый)
- ФИО бронирующего

### Список самолетов
Предустановленный список:
- Boeing 737 (150 мест)
- Airbus A320 (180 мест)
- Boeing 787 (250 мест)
- Airbus A350 (300 мест)

## API

### Рейсы
- `GET /api/flights` - получить все рейсы
- `GET /api/flights/:id` - получить рейс по ID
- `POST /api/flights` - создать рейс
- `PUT /api/flights/:id` - обновить рейс
- `DELETE /api/flights/:id` - удалить рейс

### Брони
- `GET /api/bookings` - получить все брони
- `GET /api/flights/:flightId/bookings` - получить брони для рейса
- `POST /api/bookings` - создать бронь
- `PUT /api/bookings/:id` - обновить бронь
- `DELETE /api/bookings/:id` - удалить бронь
- `POST /api/bookings/:id/transfer` - перенести бронь на другой рейс

### Самолеты
- `GET /api/aircraft` - получить список самолетов
